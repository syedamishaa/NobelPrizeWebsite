# 🏆 Nobel Prize Website

A responsive and informative web application that provides information about the Nobel Prize, its categories, notable laureates, and their remarkable achievements. The project is designed to present educational content in a clean, user-friendly, and visually appealing format.

## 🌟 Features

* 🏆 Information about Nobel Prize categories
* 👨‍🔬 Details of Nobel Prize laureates
* 📚 Educational and informative content
* 🔍 Easy navigation and structured layout
* 📱 Fully responsive design
* 🎨 Clean and modern user interface

## 🛠️ Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript
* Bootstrap


## 🎯 Project Objectives

* Provide accurate information about Nobel Prize categories and winners.
* Create an engaging educational platform.

- JavaScript Interactivity
